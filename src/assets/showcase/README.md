# Showcase creatives

Drop the creatives you want on the landing page's "Made by the pipeline"
reel **into this folder** — jpg / png / webp / gif / mp4 / webm.

- They're picked up automatically at build time (no code changes).
- Sorted by filename: prefix with `01-`, `02-`, … to control the order.
- Portrait (4:5 or 9:16) crops look best; videos autoplay muted.
- Keep files reasonably small (images < 500 KB, clips < 5 MB) — they ship
  with the site bundle. Commit + push and GitHub Actions redeploys.

Until files exist here, the section shows styled sample cards.
